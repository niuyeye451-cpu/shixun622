#### 2.1.2 NER 实体识别评估

**评估方法**：BIO 序列标注测试集，逐 token 比对，计算 Precision/Recall/F1

```
Precision = TP / (TP + FP)     # 预测为实体的token中正确比例
Recall    = TP / (TP + FN)     # 真实实体token中被正确识别的比例
F1        = 2 * P * R / (P + R)
```
