预防手段有很多，其中包括以下主要手段：

- 确定健康的生活方式，包括系好安全带、饮食健康、进行足量身体锻炼、涂防晒霜以及不抽烟等良好习惯
- 免疫接种 预防流感、肺炎球菌性肺炎和儿童期感染等感染性疾病
- 遵循 筛查 建议，以便及早发现高血压和癌症等疾病
- 如果人们出现某些疾病（如动脉粥样硬化）的风险较高或患有这类疾病，根据建议用药以防止疾病进展或加重（称为预防性药物治疗）

预防性药物治疗包括：

- 降胆固醇药物预防 动脉粥样硬化
- 阿司匹林用于在 心脏病发作 和/或 中风 高风险人群中预防这些疾病
- 他莫昔芬用于在 乳腺癌 高风险女性中预防乳腺癌
- 降压药 以降低血压并预防中风

## 健康生活方式


生活方式和疾病间有明确的关系。例如， 饮食 不健康（热量、饱和脂肪及反式脂肪酸高，以及缺乏某些其他营养物质）、不规律锻炼和吸烟可增加 心脏病 、 癌症 和 中风 的患病风险——这些疾病正是美国的主要致死原因。改变不健康的生活习惯有助于预防特定疾病和/或改善体质和生活质量。与医生和其他医疗保健专业人员交流可帮助人们做出良好决定并形成健康习惯。不过确定并保持健康生活方式仅取决于个人。始终保持健康饮食和足量锻炼对许多人来说难以做到，但做到的人可降低严重疾病的发生风险且通常感觉更佳并有很多精力。


**健康饮食习惯** 有助于人们预防或控制 高血压 、 心脏病 、 糖尿病 、 骨质疏松 和某些癌症等疾病。建议包括：

- 食用足量蔬菜、水果和全麦谷物和面包
- 减少 饱和脂肪 的摄入（如黄油、奶酪、牛肉、猪肉、带皮家禽和某些预制食品），并避免摄入反式脂肪酸
- 以更健康的脂肪替代饱和脂肪（更健康的脂肪包括油性鱼类中的脂肪 [如金枪鱼、三文鱼、鲭鱼和鲱鱼]、橄榄油和某些植物油 [如亚麻籽油、菜籽油和大豆油]，以及亚麻籽和核桃中的脂肪）
- 控制热量以将体重维持在建议水平（参见表 体重指数 (BMI))
- 限制盐的摄入量
- 获得足量钙和维生素D（饮食或补充剂中）

**身体活动和锻炼** 有助于预防 肥胖 、 高血压 、 心脏病 、 中风 、 糖尿病 、某些类型的癌症、 便秘 、 摔倒 和其他健康问题。最佳常规措施包括每周进行总计 150 分钟的中度体力活动，或每周进行 75 分钟的剧烈有氧运动（或两者结合）。运动时间至少应为 10 分钟，最好分散在整周内进行。当然，即使是少量运动也要强于完全不运动。例如，每周几次仅用 10 分钟进行身体活动也可获得很大益处，尤其是剧烈运动时。走路是一种简单而有效的运动方式，为多数人喜爱。特定运动方式也可针对特定疾病。例如，屈伸运动可提高灵活性，有助于预防 摔倒 。有氧运动可降低 心肌梗死 和 心绞痛 患病风险。


**戒烟** 对于健康的生活方式来说很重要。医生应鼓励戒烟并提供成功戒烟方法的建议，包括应用尼古丁替代品、安非他酮、伐尼克兰（有助于减少烟瘾的药物）和其他方法的信息和建议。


**安全的性行为** 很重要。安全性行为的关键是避免高危性伴侣及维持单一性伴侣制。有多名性伴侣的人在每次性行为时 正确应用乳胶安全套 可大大减少性传播感染的感染风险。对乳胶过敏的人群可使用其他类型安全套。


**限制酒精使用** 很重要。美国疾病控制与预防中心 (CDC) 建议，达到法定饮酒年龄的成人不要饮酒，或将饮酒量限制为男性每天不超过 2 杯，女性每天不超过 1 杯。（每次饮啤酒约一罐 341 mL、红酒约 142 mL、烈性酒如威士忌约 43 mL。） 即使少量饮酒也可能增加患口腔癌、食管癌和肝脏疾病等有害影响的风险。


**预防损伤** 对维持健康的生活方式至关重要。人们可以通过采取某些预防措施来降低受伤危险，例如，穿戴适当的防护装备（例如安全带）。对于老年人，以下措施可以帮助降低跌倒风险：

- 移走小地毯
- 在家中设置良好的照明
- 锻炼（尤其是改善平衡和肌肉力量）
- 定期检查视力，佩戴合适的眼镜
- 无论是添加还是更改处方，要求医生、护士或药师检查是否有可能导致跌倒的药物

**充足睡眠** 也是健康生活方式的重要组成部分，尤其会影响情绪和精神状态。睡眠不足是损伤的危险因子。


安全 101


| 采取简单的常规安全措施有助于预防受伤。下面是一些例子： 一般安全 学习 急救 。 准备或购买急救包。 学习 心肺复苏术 (CPR) 和其他缓解气道阻塞的方法，如 海姆利克氏急救法 。 骑自行车或摩托车时配戴头盔及据不同运动而异的特定防护装置，如溜旱冰或做滑板运动时使用的护腕。 安全储存枪械。 落实 游泳安全 ，不要独自游泳。 如需反复进行腕部运动（如打字），采用不易增加腕管综合征风险的姿势。 有规律并安全地进行锻炼。 戒酒或限制饮酒。 家庭安全 预防儿童跌倒及与跌倒相关的损伤： 地下室门上安装安全锁。 有儿童时关闭并锁上窗户。 换掉或盖好有锐利边缘的家具。 勿使用婴儿学步车。 安装窗户护栏，尤其二楼及以上楼层。 在楼梯顶端和底部安装楼梯门。 预防 中毒 ： 从不混合清洁产品。 烘箱和抽水马桶清洁剂、杀虫剂、酒精和防冻液保持严实密封，并置于儿童接触不到的地方。 如果家中有幼儿或有幼儿前来，将药物存放在原装瓶中，并用儿童保护性药瓶。 遵循如何安全处置过期药物和不再需要的药物的说明（见美国食品药品监督管理局网站提供的 如何处置未用药物 ）。 预防火灾： 家中每一楼板上安装可正常运转的烟雾探测器，包括地下室和每间卧室。 每月检测电池状态，并且每6个月更换新电池。 设计疏散通道并演练。 厨房内或附近放置灭火器。 请专业人员检查电气系统。 避免无人看管点燃的蜡烛。 勿在床上吸烟。 预防 一氧化碳中毒 ： 确保为室内燃烧源（如锅炉、热水器、材炉或炭炉、煤油加热器）提供充分通风。 定期清除烟道和烟囱，并检查有无漏气。 家中使用一氧化碳探测器。 预防氡暴露： 检查室内氡水平。 确保充分通风，特别是地下室。 预防 铅中毒 ： 向当地健康部门咨询并询问如何检查家庭饮用水中的铅毒性水平。 确定家中涂料是否含铅（旧宅内含有），如有任何问题，检测涂料碎屑。 检测美国以外地区生产的陶瓷餐具是否含铅。 若其医生建议，应检测儿童体内的铅水平。 预防 烧伤 ： 将热水器最高温度设为130 ° F（54.44 ° C)或更低。 食物安全 注意包装上的有效期。 易腐食品即时储存在冰箱中。 勿买有凹陷的罐装食品或盖子膨胀的任何食品。 冰箱和冷冻柜温度分别保持在40 ° F（4.44 ° C）和0 ° F（-17.78 ° C）。 冷冻2天内不食用的鲜肉（包括鱼和禽肉）。 勿让生肉汁滴到其他食物上。 做饭前后要洗手。 食物要彻底烹熟。 生肉和熟肉不应使用同一餐具和盘子。 台板、切肉板和炊具用后要用肥皂水洗净。 乘车安全 遵守限速规定并且驾驶时做好安全防护。 确保所有乘客都系好安全带。 将儿童放在汽车座椅或适合其身高和体重的其他受束装置上。 车开动时，不许将婴儿和儿童放在膝盖上 驾车前勿饮酒且勿使用非法药物或引起瞌睡的药物。 |

## 疫苗接种


疫苗 已获巨大成功。由于得到有效且安全的疫苗以及它们的广泛应用，一些危险甚至是致命的感染性疾病如白喉、百日咳、破伤风、腮腺炎、麻疹、风疹和脊髓灰质炎的感染人数较最高峰降低超过99%。此外，每花 1 美元的疫苗接种可节省 11 美元的健康医疗支出。


疫苗引起许多副作用（请参见 儿童疫苗接种问题 ）。实际副作用据不同疫苗而异，而常见副作用一般较轻微，包括肿胀、酸痛、注射部位过敏反应、偶有发烧或寒颤。也可能发生更严重的副作用。其中包括自身免疫反应（如 格林-巴利综合征 ，可引起短暂无力或麻痹）。但如适当予以疫苗接种，严重副作用非常罕见。


系统和广泛研究表明，疫苗与孤独症等其他严重副作用没有关联。疫苗引起晚期 HIV 或不育的报告是坊间传闻，没有事实依据。为避免副作用而拒绝疫苗接种会增加感染风险，而感染对健康产生的威胁远大于疫苗接种的可能副作用。

儿童和青少年、老人及免疫缺陷者通常最容易受到可通过疫苗接种预防的感染。如果这些人出现这种感染，他们通常也最有可能出现严重症状。例如，百日咳（哮咳）在婴儿中往往引起严重症状，而在其他方面保持健康的年长人群中严重程度与感冒一样为轻度。虽然对易感人群接种疫苗非常重要，但其他人群的接种也很重要。这样既可在疫苗接种后的人群中预防疾病，也可减少社区中可能出现感染进而传播给更易感人群的患者人数。因此，在社区免疫接种尽可能多的人可减少死亡和严重并发症。这种效应叫群体免疫。


## 筛查


筛查是对某种疾病高危但又无任何症状的人群进行检查（另请见 医学检查决策、筛查检测 ）。筛查可使医生及早发现疾病并及早开始治疗。早期治疗有时可防止疾病造成致命结局。例如，可诊断出宫颈或结肠异常并在癌变之前治愈。


筛查项目大大减少了一些疾病引起的死亡人数。例如，一度是美国女性最常见的癌症死因的宫颈癌，其死亡率自 20 世纪 50 年代宫颈癌筛查普及以来已下降 75%。然而，根据疾病类型、筛查的可及性与可负担性及其他因素，不同地区的下降程度有所不同。筛查还可检出不能根治的疾病，而在其产生严重后果前治疗（例如 高血压病 ）。


筛查建议通常来自政府或专业组织且基于现有的最佳研究成果，例如 美国预防服务工作组 的研究。但是，不同组织有时会提出不同的建议。有不同的建议，因为即使是最优秀的研究结果也未必具有决定性。此外，筛查建议必须考虑人们愿意接受的风险和费用，但这些因素却无法确定。因此，有关筛查的决定始终基于个体的具体情况。患者应就筛查与医生讨论，以确定最适合自己的方法。

| 某些旨在于出现症状前诊断疾病的检测（筛查检测）可能弊大于利。 |

人们可能认为任何能够检出严重疾病的测试都应该进行。筛查可以带来巨大益处。但是，它也会引起问题。例如，有时，健康患者的筛查检测结果为阳性。这些人中有一部分随后就会进行额外的随访检查和/或治疗，而这些都是不必要的，通常费用昂贵，有时还会引起疼痛或危险。


还有，有时筛查检出的异常无法治疗或不必治疗的。例如， 前列腺癌 常常生长缓慢，所以在老年男性中直到因其他原因死亡时也可能影响不到他们的健康。这种情况下，治疗本身可能比疾病更有害。另一个例子是使用全身计算机断层扫描 (CT) 来筛查每个人的癌症。通常不建议这一做法，因为其益处（如挽救生命）并不大于风险（如出现因辐射暴露引起的疾病，包括癌症）。另外，当人们被告知可能患有严重疾病时，他们常变得焦虑，而焦虑会影响健康。


基于这些，仅建议如下情况进行筛查：

- 人们确实存在出现疾病的风险。
- 筛查的查验精准。
- 某疾病在症状出现前被诊断可进行更有效治疗。
- 合理筛查的医疗保健效费比更高。

一些筛查（如针对宫颈癌和结肠癌的检查）应当建议某一年龄或某一性别的所有人进行。对于因其他风险因素导致风险增加的人，建议早于一般风险人群更早或更频繁地进行检查，或者可能会建议进行额外的检查。例如，如果患者有结直肠癌家族史或患有一种增加 结直肠癌 患病几率的疾病（如 溃疡性结肠炎 ），建议筛查性结肠镜检查频率要大于针对一般风险人群的正常频率。如果女性有近亲患有乳腺癌（ 家族史 ），除了乳房 X 线摄影外，还建议通过磁共振成像 (MRI) 筛查乳腺癌。


一些筛查措施被推荐给患有特定疾病的人们。例如，糖尿病患者应每天检查至少一次足部是否发红和有溃疡，如果忽视，可能导致严重感染并最终截肢。


## 预防性药物治疗


预防性药物治疗（亦称化学性预防）是应用药物预防疾病。推荐这种治疗，一定是针对高危人群的疾病预防，同时所推荐药物引起副作用的风险较低。


预防性药物治疗益处明确，例如在特定人群（如晚期 HIV）中预防感染，在偏头痛患者中预防头痛和其他许多特定疾病。尽管预防性药物治疗只是针对特定疾病，但由于有些疾病很常见，这种治疗对很多人很有用。例如在冠心病和中风高危人群，阿司匹林常规推荐使用。新生儿常规接受滴眼药，防止眼睛淋球菌感染。乳腺癌高危女性可从预防性药物治疗中受益（如服用他莫昔芬）。


三级预防


| 三级预防分别是初级预防、二级预防和三级预防。 在 初级预防 中，实际为预防疾病出现。 初级预防包括以下几类： 疫苗接种 建议改变高风险行为 有时进行化学预防 二级预防 ，发现疾病并在出现症状和之前早期治疗，减少严重并发症发生。 二级预防包括以下几类： 筛查项目，如使用乳房 X 光摄影术检测乳腺癌和使用双能 X 光吸收测量法 (DXA) 检测骨质疏松。 追踪被确诊为性传播感染患者的性伴侣（接触者追踪），并在必要时对其加以治疗以尽可能减少疾病传播。 三级预防 ，是对已经存在的慢性疾病的预防，阻止并发症或危害加重。 三级预防包括以下几类： 对于糖尿病患者：控制血糖、良好的皮肤护理、经常检查足部、以及经常锻炼，以防发生心血管疾病 对于中风患者：适用阿司匹林以防止第二次中风发生 提供支持和复健措施，以防止病情恶化和优化生活质量，如损伤、心肌梗死或中风的康复 预防残疾患者出现并发症，如防止卧床患者发生褥疮。 |

| 癌症 | 不吸烟（预防肺癌和许多其他癌症）。 保持富含纤维、水果和蔬菜并限制脂肪（尤其是饱和脂肪和反式脂肪酸）和热量的平衡饮食（预防乳腺癌和结直肠癌）。 减少盐或烟熏食物的摄入量（预防胃癌）。 避免过度日光暴露并使用高防晒系数的防晒霜（预防皮肤癌）。 确保儿童接种 HPV（人乳头状瘤病毒）疫苗（预防宫颈癌和喉癌）。 考虑服用推荐的预防癌症的药物（如针对乳腺癌高风险女性的他莫昔芬）。 接受推荐的筛查性检查 |
| 慢性阻塞性肺疾病（COPD） | 不吸烟。 避免暴露于有毒物质（尤其是在工业环境中）。 |
| COVID-19 | 如果医生建议，每年接种 COVID-19 疫苗 |
| 糖尿病 （2 型） | 经常锻炼。 平衡饮食。 保持建议的体重。 通过饮食保持正常血脂水平，必要时使用药物。 通过饮食、锻炼、减压保持正常血压，必要时使用药物。 不吸烟。 |
| 心脏病 和 中风 | 通过饮食和药物（如有必要）保持正常的胆固醇和其他脂肪（脂质）水平。 通过饮食、锻炼、减压保持正常血压，必要时使用药物。 保持富含纤维、水果和蔬菜并限制脂肪（尤其是饱和脂肪和反式脂肪酸）和热量的平衡饮食。 保持建议的体重。 不吸烟。 有规律锻炼，包括有氧锻炼（如快步走、骑自行车和慢跑）以及肌肉加强锻炼（如自由重量训练或使用重量器材）。 如有建议则服用阿司匹林和降脂药物（针对大多数有冠状动脉疾病高风险的成人）。 请勿使用可卡因或其他非法兴奋剂。 |
| 高血压 | 保持富含纤维、水果和蔬菜并限制盐、脂肪（尤其是饱和脂肪和反式脂肪酸）和热量的平衡饮食。 有规律锻炼，包括有氧锻炼以及肌肉加强锻炼。 通过饮食、锻炼和药物（如有必要）保持正常胆固醇水平。 保持建议的体重（通过饮食和锻炼）。 不吸烟。 |
| 流行性感冒 | 每年接受流感疫苗接种（尤其是婴儿、老年人和有心脏病、肺病或免疫系统疾病的人）。 |
| 肝脏疾病 | 如须饮酒，要适量。 接种甲肝和乙肝疫苗（所有儿童以及对于该疾病有高风险因素的成人）。 |
| 骨关节炎 | 进行肌肉加强和伸展锻炼。 保持积极的身体活动。 保持建议的体重。 |
| 骨质疏松症 | 摄入足量钙和维生素D（饮食或补充剂中）。 负重锻炼（如步行、慢跑、网球和舞蹈），每天至少30分钟。 服用医生开具的增强骨质的药物。 限制咖啡因和酒精的摄入（一天摄入量限为一次饮量†）。 不吸烟。 |
| 肺炎 | 接种 肺炎球菌疫苗 （有两种疫苗有助于预防肺炎，对接种者建议各不相同） |
| 性传播感染 | 节制性行为或限制性伴侣数。 使用避孕套，进行安全性行为。 11-12 岁的儿童（或 9 岁以下）以及所有 26 岁或 26 岁以下且未接种的人群，应接种 HPV 疫苗。 如果年龄在 27 至 45 岁之间，请向医生咨询获益。 根据性行为，与医生讨论是否需要接受预防 HIV 感染的药物。 |
| 牙齿脱落 | 定期刷牙和使用牙线。 限制糖果的食用次数。 定期看牙医。 如果需要，补充氟化物（例如，6 个月以上的学龄前儿童的饮水源中缺少氟化物）。 |
| * 除上述预防措施外，人们还应咨询医生以了解其他推荐的筛查检测项目。 |
| †1 次饮量 = 一罐 341 mL 的啤酒、142 mL 的红酒或 43 mL 的烈性酒（如威士忌）。 |
| CDC = 美国疾病控制和预防中心；HPV = 人乳头瘤病毒。 |


* 除上述预防措施外，人们还应咨询医生以了解其他推荐的筛查检测项目。


†1 次饮量 = 一罐 341 mL 的啤酒、142 mL 的红酒或 43 mL 的烈性酒（如威士忌）。


CDC = 美国疾病控制和预防中心；HPV = 人乳头瘤病毒。
